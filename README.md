# MusiCluster

musiCluster is an application that performs API requests to the lastfm server and stores responses into a mySQL database.

View deployed application here: https://musicluster.herokuapp.com/

<h2>Technologies Used:</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
  <li>Axios</li>
  <li>Bcrypt</li>
  <li>Body-parser</li>
  <li>Crypto</li>
  <li>dotenv</li>
  <li>Express</li>
  <li>Handlebars</li>
  <li>Session</li>
  <li>Multer</li>
  <li>mySQL</li>
  <li>Node.js</li>
  <li>Nodemon</li>
  <li>Passport</li>
  <li>lastfm</li>
  <li>Sequelize</li>
</ul>
